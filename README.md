# 📊 Data Professional Survey Analysis Dashboard

## 📌 Project Overview
This project provides an in-depth exploratory data analysis of a **Data Professional Survey**. Using **Microsoft Power BI**, I transformed raw survey data into an interactive dashboard that uncovers key trends regarding demographics, compensation, job satisfaction, and technology preferences across various data roles.

---

## 💡 Key Insights Discovered
Based on the survey of **630 data professionals**, the dashboard reveals the following insights:

* **Demographics:** The average age of professionals in the field is **29.87 years**. The majority of survey takers are located in the **United States**, followed by India, the UK, and Canada.
* **Technology Preferences:** **Python** completely dominates as the favorite programming language across almost all data roles (Data Analysts, Data Scientists, Data Engineers), leaving R, C/C++, and Java far behind.
* **Work/Life Balance:** The average happiness score regarding work/life balance is **5.74 out of 10**, indicating moderate satisfaction but leaving significant room for improvement in the tech industry.
* **Compensation & Roles:** Average salaries show a close distribution between genders (~$53.5k - $55.1k). **Data Analyst** is the most common current role among the respondents.

---

## 🛠️ Tools & Techniques Used
- **Microsoft Power BI:** Core tool used for building the dashboard.
- **Power Query:** Used for data cleaning, transformation, and shaping the raw survey responses.
- **Data Modeling:** Establishing relationships between different survey attributes.
- **Data Visualization:** Creating interactive elements like Treemaps, Donut charts, Gauge charts, and Stacked Bar charts.
